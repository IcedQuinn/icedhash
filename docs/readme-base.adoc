= Iced Quinn's Thing Hasher

`icedhash` is a collection of cryptographic and non-cryptographic hashing routines which have been ported to native Nim.

== Dependencies

For end users:

 - None!

For people hacking on icedhash:

 - https://lets-cli.org/[Lets] (optional, top-level task runner)
 - https://github.com/jmmv/kyua[kyua] (optional, runs TAP tests.)
 - https://docs.asciidoctor.org/[asciidoctor] (optional, to regenerate the README.)
 - https://pandoc.org/[pandoc] (optional, to regenerate the README.)
 - icedbintext. For converting hashes to hexadecimal outputs.
